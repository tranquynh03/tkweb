Các Thẻ HTML Cơ Bản

1. HTML Basic

Ý nghĩa thẻ: Đây là phần giới thiệu cơ bản về HTML, cho phép bạn tạo cấu trúc cơ bản cho một trang web.

<html>
<head></head>
<body></body>
</html>

2. HTML Elements

Ý nghĩa thẻ: Các phần tử HTML là các thành phần trong tài liệu HTML.

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

3. HTML Attributes

Ý nghĩa thẻ: Các thuộc tính trong HTML cung cấp thông tin bổ sung cho phần tử.

<a href="https://www.example.com">Click here</a>

4. HTML Headings

Ý nghĩa thẻ: Các thẻ <h1> đến <h6> được sử dụng để tạo các tiêu đề trong HTML.

<h1>Main Heading</h1>
<h2>Subheading</h2>

5. HTML Paragraphs

Ý nghĩa thẻ: Thẻ <p> dùng để tạo đoạn văn trong HTML.

<p>This is a paragraph of text.</p>

6. HTML Styles

Ý nghĩa thẻ: Thẻ <style> trong phần <head> được sử dụng để thêm các quy tắc CSS cho trang web.

<style>
    body { background-color: lightblue; }
    h1 { color: red; }
</style>

7. HTML Formatting

Ý nghĩa thẻ: Các thẻ HTML để định dạng văn bản.

<b>Bold text</b>
<i>Italic text</i>
<u>Underlined text</u>

8. HTML Quotations

Ý nghĩa thẻ: Thẻ <blockquote> được sử dụng để trích dẫn văn bản dài, còn thẻ <q> được sử dụng cho trích dẫn ngắn.

<blockquote>This is a long quote</blockquote>
<q>Short quote</q>

9. HTML Comments

Ý nghĩa thẻ: Thẻ <!-- --> được sử dụng để thêm chú thích trong mã HTML mà không làm ảnh hưởng đến giao diện của trang.

<!-- This is a comment -->

10. HTML Colors

Ý nghĩa thẻ: HTML cho phép bạn định nghĩa màu sắc của văn bản, nền và các yếu tố khác trong trang web.

<p style="color: blue;">This is a blue paragraph.</p>

11. HTML Links

Ý nghĩa thẻ: Thẻ <a> được sử dụng để tạo liên kết đến các trang web khác hoặc các vị trí khác trong cùng một trang.

<a href="https://www.example.com">Visit Example</a>

12. HTML Images

Ý nghĩa thẻ: Thẻ <img> dùng để chèn hình ảnh vào trang web.

<img src="image.jpg" alt="Example Image">

13. HTML Tables

Ý nghĩa thẻ: Các thẻ <table>, <tr>, <th>, và <td> được sử dụng để tạo bảng trong HTML.

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
    </tr>
</table>

14. HTML Lists

Ý nghĩa thẻ: HTML hỗ trợ hai loại danh sách: danh sách không có thứ tự (<ul>) và danh sách có thứ tự (<ol>).

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

15. HTML Block & Inline

Ý nghĩa thẻ: Thẻ block-level chiếm toàn bộ chiều rộng của dòng và bắt đầu một dòng mới, còn thẻ inline chỉ chiếm không gian cần thiết.

<div>This is a block-level element</div>
<span>This is an inline element</span>
